Day 5: Linux Terminal Trainer-Now aligned to LPIC-1 certification

Big update today. I realized building a terminal trainer without a clear goal is like studying without a syllabus. So I aligned the entire app to LPIC-1 exam objectives.

What changed:
- 12 guided lessons mapped to real LPIC-1 topics (103.1 through 105.1)
- Each lesson shows the exam objective code so you always know what you're studying for
- 18 new commands: grep, head, tail, wc, sort, chmod, find, pipes, redirects, and more
- A capstone lesson where you investigate a security incident using everything you learned

The idea: when you finish all lessons, you should feel comfortable with the hands-on portions of the LPIC-1 101 exam. Practice exam mode coming next.

Built with React + TypeScript. Try it live: https://linux-terminal-trainer-psi.vercel.app
